Welcome to Paithon

Paithon is a groundbreaking programming language based entirely on Python but with its own identity. Paithon provides the same syntax and functionality as Python, making it the perfect choice for those who want a Python-like experience with a unique twist.

How to Install Paithon

1. Download the files.
2. Open Command Prompt.
3. Run Install-Paithon.bat.
4. Restart Command Prompt and type 'paithon' to run your scripts.

How to Uninstall Paithon

1. Open Command Prompt.
2. Run Uninstall-Paithon.bat.
3. Restart Command Prompt.
